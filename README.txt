Sujet : Réalisation d’un jeu simple sur unity à base de grille hexagonale.

Version Unity : 2017.1.0p5
Build fournis : Windows x64 et Android

Nouveau élément de gameplay ajoutée : aggrandissement de la zone de jeu dans le temps.

Plugin : DOTWEEN (http://dotween.demigiant.com/documentation.php)
Ressource : Hexagon coordonnées axial : http://www.redblobgames.com/grids/hexagons/
